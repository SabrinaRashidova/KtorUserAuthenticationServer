This project is a simple backend built with **Ktor**, following the **Clean Architecture** structure, and using **Exposed ORM** with **PostgreSQL**.  
It contains basic authentication features: **register**, **login**, **update password**, and **delete user**.

---
Technologies Used

- **Kotlin**
- **Ktor Server**
- **Exposed ORM**
- **PostgreSQL**
- **BCrypt (password hashing)**
- **dotenv (environment variables)**
